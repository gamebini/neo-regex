전문가 검증
^[a-zA-Z0-9.!#$%&'*+/=?^_`{|}~-]+@[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?(?:\.[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?)*$

강력한 비밀번호

전문가 검증
^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{12,}$

최소 12자, 대소문자, 숫자, 특수문자 포함 강력한 비밀번호 검증

복잡도: 중간 성능: 좋음 보안: 높음

국제 전화번호

전문가 검증
^\+?[1-9]\d{1,14}$

E.164 국제 표준을 준수하는 전화번호 형식 검증

복잡도: 낮음 성능: 우수 국제표준: E.164

JSON 구문 파싱

전문가 추출
"(?:[^"\\]|\\.)*"\s*:\s*(?:"(?:[^"\\]|\\.)*"|-?\d+(?:\.\d+)?(?:[eE][+-]?\d+)?|true|false|null)

JSON 객체의 키-값 쌍을 정확하게 추출하는 고급 패턴

복잡도: 높음 정확도: 98% JSON 호환

IPv4/IPv6 주소

전문가 추출
(?:(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)|(?:[0-9a-fA-F]{1,4}:){7}[0-9a-fA-F]{1,4}

IPv4와 IPv6 주소를 모두 정확하게 추출하는 통합 패턴

IPv4/IPv6 지원 정확도: 99.9% RFC 호환
